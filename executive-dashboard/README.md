# Executive Weekly Briefing Dashboard

An interactive HTML executive dashboard featuring a filter bar, clickable badge filtering, collapsible card sections, and a live item count.

**▶ View the live demo:** https://robertciceroson.github.io/process-engineering-portfolio/executive-dashboard/
